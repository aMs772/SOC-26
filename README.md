In this branch, I have built a small neural network of 3 layers for identifying handwritten digits using ml.
I have used MNIST data set and learned from michael nielsen, 3blue1brown.

input layer: 784 neurons
middle layer: 30 neurons
output layer: 10 neurons

input layer takes the the scanned image of digit (28*28 pixels) and each pixel corresponds to a neuron in the input layer
output layer has 10 neurons each corresponding to a digit from (0-9)

the model accuracy after 30 iterations is around 96%
